## Summary
Python code using Pandas for analysis of Sony's stock value over the last year (2025). 
This analysis was completed using [data from Nasdaq](https://www.nasdaq.com/market-activity/stocks/snejf). 

## Output
This code outputs a line graph, showing how the closing value ($) of Sony's stocks have changed over the last year.      

## Execution
Can be run using a normal VSCode setup or Python interpreter.
